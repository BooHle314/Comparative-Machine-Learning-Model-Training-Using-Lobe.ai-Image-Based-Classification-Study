# Comparative-Machine-Learning-Model-Training-Using-Lobe.ai-Image-Based-Classification-Study
This project explores the use of Lobe.ai, to train a model for image-based classification of objects, scenes, or actions. The objective is to understand how user-friendly AI tools simplify the machine learning workflow, while also evaluating their strengths and limitations in comparison to other platforms such as Teachable Machine.

Objective
Train a machine learning model using Lobe.ai’s drag-and-drop interface
Understand the impact of data collection quality and diversity on model performance
Evaluate model accuracy using built-in Lobe.ai tools
Compare Lobe.ai vs Teachable Machine in terms of usability and functionality
Identify real-world applications of low-code AI systems

#Data Collection
Collected images representing different classes of objects/actions/scenes
Ensured variation in:
Lighting conditions
Background environments
Angles and perspectives
Built a balanced dataset to improve model generalization

#Model Training (Lobe.ai)
Imported dataset into Lobe.ai platform
Utilized drag-and-drop interface for model training
Trained a supervised image classification model
Iteratively refined dataset based on performance feedback

#Testing & Evaluation
Tested model using unseen images
Evaluated performance using Lobe.ai’s built-in accuracy metrics
Identified misclassifications and improved dataset where necessary

#Comparison: Lobe.ai vs Teachable Machine
Feature	Lobe.ai	Teachable Machine
Interface	More structured & professional	Very beginner-friendly
Customization	Slightly more advanced	Limited
Evaluation tools	Built-in performance insights	Basic evaluation
Ease of use	Moderate	Very easy
Flexibility	Better for scalable projects	Best for quick experiments

#Key Learnings
Machine learning performance is highly dependent on dataset quality and variation
Low-code AI tools significantly reduce the complexity of model training
Lobe.ai provides better visualization of model performance compared to basic tools
Comparing tools helps understand trade-offs between simplicity and control in AI systems
